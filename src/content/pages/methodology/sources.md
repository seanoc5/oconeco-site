# Source agency directory

FAND deliberately cherry-picks a small number of key indicators from each source agency. The point is to let users **leave the site to verify** — and come back. Every number on every page links to its source agency, table, year, and URL.

## International sources (country-level data)

| Agency | What we cherry-pick |
|---|---|
| **WDI** (World Bank Open Data) | GNI Atlas, GDP, population, education indicators |
| **IMF (IIP / IFS)** | International Investment Position, balance-of-payments stocks |
| **FAO / FAOSTAT** | Agricultural land, forestry, fisheries |
| **USGS Minerals** | 31 mineral commodities, reserves and production |
| **EIA / Energy Institute** | Fossil fuel reserves and production |
| **OECD Fixed Assets** | Produced assets, capital stock, depreciation |
| **UN Population Division (WPP)** | Population, working-age share, life expectancy |
| **BIS** | Bank locational statistics |

## US sources (state and county data)

| Agency | What we cherry-pick |
|---|---|
| **BEA** | GDP, income, FDI position |
| **BLS** | Wages by industry, QCEW |
| **Census Bureau** | ACS demographics, education attainment |
| **EIA** | Energy production by state |
| **USGS** | Mineral deposits by state and county |
| **NOAA** | Fisheries, coastal economy |
| **USDA NASS** | Cropland, rental rates, agricultural production |
| **IHME / CHR** | Life expectancy at state and county level |

## The UX pattern

Every data value on a country / state / county page should have a small source icon or footnote. Click it → see the agency, table name, year, and a direct URL to the original data. **Not a separate bibliography page** — inline, contextual, and immediate.

The four-axis architecture treats *Source* as a first-class navigation axis: every number on the site has a verifiable provenance chain back to a public agency.
